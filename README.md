<h1><b>Horsebot</b></h1>
Horsebot is the official Stallion Squad <a href="https://discord.gg/JYmVCVmSR7">Discord</a> Bot that serves as a companion for players. Can be used to retrieve players Steam inventories, item information and wiki data on the fly!

### Installation
<ol>
  <li>Extract files into a folder</li>
  <li>Open a terminal in your folder and install dependencies by using</li>
  
  `npm i`
  <li>To start use</li>
  
  `node index.js`
</ol>

### Creating environment variables
<ol>
  <li>Create .env file in your folder</li>
  <li>Define enviromental variables like that:</li>

```
BOT_TOKEN=YOUR_BOT_TOKEN
STEAM_API_KEY=YOUR_STEAM_API_KEY
```
  
   </ol>
